Build Setup
James Spratt, 2025 Apr 07

Development of The Mirror is targeting Microsoft Windows and Linux so far as the main platforms. I don't have plans or the means to support OSX, but I might consider that in the future.

I don't have a sophisticated setup, just a single machine dual-booting both Windows 10 and Debian Linux. Each OS has access to an NTFS formatted 2TB drive which has a folder on it called SHARED_ACCESS. Within this folder are a number of git repositories, one of them being the main repository for The Mirror. Each OS has a local copy of the repos that they both pull from.

This solution works for now, I could even move the repo to the cloud or even Github/Gitlab, but for now, I'm just keeping it on my local machine. I do my own personal backups so I'm not concerned about losing data. Also, if anyone else wants to work on the code then it won't be hard to move it to the cloud. Nothing about my workflow will change, just the repo will be in a different location.

Orginally I was just manually syncing the files between each OS by using rsync, but that was just temporary before I started using a proper solution. git is a great tool and I think every programmer should be familiar with it, although limiting yourself to a restricted subset of the features is probably best.

X11 & GLX

To talk about opening a native window on any platform we need to understand what the native windowing system is. In the case of Microsoft *Windows*, the windowing system is built into the general platform API known as win32. In the case of linux, windowing systems are seperate from the kernel (properly speaking, seperate from linux itself), but the most widely used windowing system across different linux distributions is X11 by far. Wayland is another option, but I won't be touching that.

Anyways, start with this C code, and name it something like "x11_window.c":

#include <stdio.h> #define TRUE 1 #define FALSE 0 int main() { printf("Hello World!\n"); return 0; }

then, compile with this command:

gcc x11_window.c -o x11_window

run it with:

./x11_window

and you should get "Hello World!" as output.

Also, at the end of the main function add a cleanup section like this:

#include <stdio.h> #define TRUE 1 #define FALSE 0 int main() { printf("Hello World!\n"); CLEANUP_AND_EXIT: return 0; }

And that's that
