What this Blog is

Currently, this blog is a test site for me to learn HTML and CSS. I'm going to use this blog to learn some basic web development, and then I'm going to actually turn this into my blog where I will aim to make a post weekly on either a new topic, or an update on a project I am working on.

Main Goals for the Future

Eventually I want this blog to turn into something more serious, once I learn enough web development. My goals for this blog will be the following, ranked somewhat in order of importance:

  • A place where I post updates on the current project(s) that I am working on.
  • A portfolio of all my main accomplishments in my career.
  • A personal repository of documentation for all of the useful knowledge that I have picked up in my career, mostly programming and computer science related.
  • A collection of rogramming tips and techniques
  • A bunch of personal rants on uncategorized topics such as philosophy, science, rhetoric, politics, etc.
