Blog of James Spratt

Welcome to my personal blog/website! I mostly write about programming, general computer science, and mathematics, but I also occassionally write about other random topics.

Right now the blog is a testing ground for my knowledge of basic web development (HTML, CSS, JS), but I'm mainly hoping to use it as repository for all of the main projects that I am working on. I also want to make blog posts on techniques and little discoveries that I make that are related to math, computer science, and the art of programming. And there will also be just general topic blog posts on any paticular subject that I feel like writing about. It could be philosophy, science, politics, history, etc.
I'm also planning on including some certain sections that don't necessarily correspond to a "post". For example, I have a section called "important resources" which is basically just a large listing of important resources that I think are very useful to anyone in similar line of work as myself (or just in general).

This blog will also function as a "personal repository" of the information that I have learned over the years. Sometimes I wonder if I have lost a significant amount of useful knowledge just because I didn't take the time to document it properly so that I could refer back to it later.
So, I'm hoping that with this blog, I'll be able to document all of the useful information and knowledge that I aquire so that I can share it not just publicly, but also with my future self.

Ideally, I'd like to update this blog once a week by making a new blog post every Sunday, or at least every other Sunday if not every week. Either I'll be posting a blog about something completely new, or I'll update a current project or blog thread that I've previously been working on (or both if I'm feeling ambitious). We'll see if this actually happens, but I really want to be diligent about this because I see this blog as a tool to keep myself on-track and focused on my work. The way I see it, if I don't update this blog weekly, then it means that I'm slacking on my projects and work.
